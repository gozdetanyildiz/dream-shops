# Dream Shops 🛒

Dream Shops is a Spring Boot based e-commerce backend REST API.

## 🚀 Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven

## 📦 Features
- Product & Category CRUD
- User management
- Cart & Order management
- DTO mapping
- Global exception handling

## ⚙️ Run the Project

```bash
git clone https://github.com/gozdetanyildiz/dream-shops.git
cd dream-shops
mvn spring-boot:run

Application runs at:

http://localhost:9191
